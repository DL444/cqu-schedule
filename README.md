# CQU Schedule
[CQU Schedule](https://schedule.dl444.net) 可以获取重庆大学课程信息，并生成相应的 iCalendar 日历订阅或 ICS 日历文件。

只需登录帐户即可生成订阅，无需手动导出 Excel 或提取 Cookie 等。

日历订阅支持自动更新。
